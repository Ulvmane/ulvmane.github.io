# Project Overview

This directory contains a static portfolio website for Felix Ulvmåne, a Tools and Engine Programmer. The website serves to showcase his work, primarily in game development, and provides information about him and links to his Résumé and GitHub profile.

## Directory Overview

The project is structured as a collection of HTML, CSS, and image files. There are no backend components or complex build processes.

*   `.html` files: These are the core pages of the website, including the homepage (`index.html`), an about page (`about.html`), and dedicated pages for various game projects within the `games/` directory.
*   `portfolio.css`: This file contains all the styling rules for the entire website.
*   `___content/`: This directory stores promotional images used across the project pages.
*   `felix_minifig.png`, `felix_minifig_2.png`: These are likely profile pictures or branding elements for the website owner.
*   `FelixUlvmane_CV_2023.pdf`: This file contains Felix Ulvmåne's Résumé.

## Key Files

*   `index.html`: The main entry point of the website, displaying featured projects.
*   `about.html`: Provides information about Felix Ulvmåne.
*   `games.html`: A page listing various game projects.
*   `projects.html`: A page for other projects.
*   `portfolio.css`: Defines the visual style and layout of the website.
*   `FelixUlvmane_CV_2023.pdf`: The owner's curriculum vitae.

## Usage

This is a static website designed to be served directly by a web server (e.g., Apache, Nginx) or hosted on platforms like GitHub Pages. To view the website locally, simply open `index.html` in a web browser. Navigation is handled through the internal links within the HTML files.
